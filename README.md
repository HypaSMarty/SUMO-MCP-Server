## 🛠️ Overview

SUMO-MCP 是一个连接大语言模型 (LLM) 与 Eclipse SUMO 交通仿真的中间件。通过 Model Context Protocol (MCP)，它允许 AI 智能体（如 Claude, Cursor, TRAE等）直接调用 SUMO 的核心功能，自动完成从OpenStreetMap 数据获取、路网生成、需求建模到仿真运行与信号优化的全流程。

## 📋 Features

- **简单配置**: 轻松设置与运行SUMO-MCP-Server。
- **全面集成**: 支持多种AI智能体，增强模拟体验。
- **自动化流程**: 从数据获取到信号优化，全面支持。
- **用户友好**: 设计考虑非技术用户，便于操作。

## 🖥️ System Requirements

- **操作系统**: Windows 10 或 macOS 10.14 及以上版本
- **内存**: 至少 4 GB RAM
- **硬盘空间**: 需要 1 GB 可用空间
- **网络**: 需要互联网连接以获取最新的数据和功能
